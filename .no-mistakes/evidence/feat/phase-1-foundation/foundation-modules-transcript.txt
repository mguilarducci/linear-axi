──────── context.ts — resolveLinearContext / requireApiKey ────────
resolveLinearContext() => {"apiKeyPresent":true,"apiKey":"lin_api_demoKey_abc123","teamKey":"ENG"}
requireApiKey(ctx) => lin_api_demoKey_abc123
requireApiKey(no key) throws AxiError code=AUTH_REQUIRED suggestions=["Set LINEAR_API_KEY to a Linear personal API key, then retry","Create one at Linear → Settings → Security & access → Personal API keys"]

──────── linear.ts — real request, Authorization header captured over the wire ────────
linearRequest returned data => {"viewer":{"id":"u_123","name":"Demo User"}}
Authorization header the SERVER actually received => "lin_api_demoKey_abc123"
  → equals the raw key, NO "Bearer " prefix: true
  → starts with "Bearer "? false

──────── linear.ts — GraphQL auth error maps to a structured AxiError ────────
linearRequest(auth fail) throws code=AUTH_ERROR message="Your API key is invalid."
  suggestions => ["Check that LINEAR_API_KEY is set to a valid Linear personal API key","Create a key at Linear → Settings → Security & access → Personal API keys"]

──────── linear.ts — missing key fails fast WITHOUT any network call ────────
linearRequest(no key) throws code=AUTH_REQUIRED (server header unchanged: true)

bare HTTP 503  => code=SERVER_ERROR  message="Linear API server error (HTTP 503)"

──────── body.ts — cleanBody / truncateBody / takeBody ────────
